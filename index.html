Casos de Estudio

Análisis de Sentimiento y Tópicos (NLP)

El Reto

Extraer insights accionables a partir de miles de reviews de texto no estructurado para entender las preocupaciones clave y el sentimiento general de los usuarios de una plataforma.

La Solución

Desarrollé un pipeline de NLP en Python que limpia, procesa y analiza el texto. Apliqué Topic Modeling (LDA) para descubrir temas ocultos y análisis de sentimiento para cuantificar la opinión.

El Impacto

Se entregó un dashboard interactivo que visualiza los principales tópicos de queja (ej. "buffering", "precio") y el sentimiento asociado, proveyendo una guía clara para la priorización de mejoras del producto.

NLP Gensim Topic Modeling
VER NOTEBOOK

Servicio de Predicción con Orquestación

El Reto

Construir un servicio de predicción de alto rendimiento que pudiera manejar tareas pesadas (como el re-entrenamiento) de forma asíncrona y fuera fácil de gestionar en un entorno de desarrollo multi-servicio.

La Solución

Diseñé una arquitectura con FastAPI para la API, Celery para la gestión de tareas asíncronas en background y Docker Compose para orquestar los servicios (API, worker, broker) de forma unificada.

El Impacto

La API mantiene una latencia baja para las predicciones mientras los trabajos de entrenamiento se ejecutan de forma aislada, creando un sistema robusto, escalable y listo para ser migrado a Kubernetes.

Deep Learning Feature Engineering Arquitectura Modular FastAPI Docker Compose Celery
VER ARQUITECTURA

Dashboard Interactivo para Business Intelligence

El Reto

Traducir datos complejos de negocio en insights accionables y fácilmente consumibles por stakeholders no técnicos para agilizar la toma de decisiones estratégicas.

La Solución

Desarrollé un dashboard interactivo en R Shiny y Power BI, enfocado en el storytelling de datos, permitiendo la exploración de KPIs y la simulación de escenarios en tiempo real.

El Impacto

Se empoderó a los equipos de negocio con una herramienta de auto-servicio que redujo la dependencia del equipo de datos y mejoró la comprensión de las métricas clave.

R Shiny Power BI KPI's & Storytelling
